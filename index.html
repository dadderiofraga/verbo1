Futuro do Presente do Indicativo

Terminações dos Verbos Regulares

1ª Conjugação (-ar)

Pessoa Estrutura Exemplo (Falar)
(Eu) radical + -arei falarei
(Tu) radical + -arás falarás
(Ele) radical + -ará falará
(Nós) radical + -aremos falaremos
(Vós) radical + -areis falareis
(Eles) radical + -arão falarão

2ª Conjugação (-er)

Pessoa Estrutura Exemplo (Beber)
(Eu) radical + -erei beberei
(Tu) radical + -erás beberás
(Ele) radical + -erá beberá
(Nós) radical + -eremos beberemos
(Vós) radical + -ereis bebereis
(Eles) radical + -erão beberão

3ª Conjugação (-ir)

Pessoa Estrutura Exemplo (Partir)
(Eu) radical + -irei partirei
(Tu) radical + -irás partirás
(Ele) radical + -irá partirá
(Nós) radical + -iremos partiremos
(Vós) radical + -ireis partireis
(Eles) radical + -irão partirão

Padrão Geral de Formação

Futuro do Presente = Verbo no Infinitivo + Terminações do verbo "haver" no Presente

Exemplos de Uso

Estudarei para a prova amanhã. (Ação futura certa)
Falaremos com o diretor na próxima semana. (Plano)
Partiremos assim que você chegar. (Ação condicionada)
Beberei mais água este ano. (Promessa)
